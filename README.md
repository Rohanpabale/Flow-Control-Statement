# 🚀 Flow Control Statements in Java

This repository contains my practice programs on **Flow Control Statements in Java**.
Flow control statements are used to control the execution flow of a program.

---

## 📂 Topics Covered

### 🔹 Conditional Statements

* if, if-else
* else-if ladder
* switch-case

### 🔹 Looping Statements

* for loop
* while loop
* do-while loop
* Nested loops

### 🔹 Jump Statements

* break statement
* continue statement
* return statement
